# Analyse des Ventes Superstore (Python & SQL)

Ce projet combine l'analyse exploratoire avec **Python** et la manipulation de données complexes avec **SQL** pour extraire des insights business d'un dataset de vente.

## Stack Technique
- **Python** (Pandas, Seaborn, Matplotlib) : Pour le nettoyage, la visualisation et l'analyse de Pareto.
- **SQL** : Pour l'extraction de KPIs, les fonctions fenêtres (Ranking) et l'analyse de profitabilité.
- **Power BI** : Les données nettoyées sont exportées pour un dashboard final.

## Résultats Clés
- **Règle des 80/20 :** 15% des produits génèrent 80% du chiffre d'affaires.
- **Impact Remises :** Les remises supérieures à 20% entraînent systématiquement des pertes.
- **Top Catégorie :** La technologie est le segment le plus rentable malgré un volume de vente moyen.

## Comment utiliser ce repo
1. Explorez les requêtes SQL dans `/sql` pour les KPIs.
2. Consultez le notebook `/notebooks` pour le storytelling visuel.
